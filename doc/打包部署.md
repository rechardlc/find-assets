# 打包部署

## 1. 环境要求

| 项目 | 要求 |
|------|------|
| 编译环境 | Go 1.25+ |
| 运行环境 | **零依赖**，无需 Go / Python / 数据库 |
| 网络 | 需能访问东方财富 push2 接口 |

---

## 2. 本地打包（Windows）

在项目根目录执行：

```powershell
cd c:\Users\33002\Desktop\github\find-assets

# 推荐：压缩体积
go build -ldflags="-s -w" -o 选股器.exe ./cmd/scanner
```

参数说明：

| 参数 | 含义 |
|------|------|
| `-ldflags="-s -w"` | 去掉符号表和调试信息，体积约 10~15 MB |
| `-o 选股器.exe` | 输出文件名 |
| `./cmd/scanner` | 主程序包路径 |

---

## 3. 交叉编译

### Linux amd64

```powershell
$env:CGO_ENABLED="0"
$env:GOOS="linux"
$env:GOARCH="amd64"
go build -ldflags="-s -w" -o dist/scanner ./cmd/scanner
Remove-Item Env:GOOS, Env:GOARCH, Env:CGO_ENABLED
```

Linux 运行：

```bash
chmod +x scanner
./scanner -mode=day
```

### macOS arm64

```powershell
$env:CGO_ENABLED="0"
$env:GOOS="darwin"
$env:GOARCH="arm64"
go build -ldflags="-s -w" -o dist/scanner-darwin-arm64 ./cmd/scanner
```

## 4. 一键打包脚本（可选）

创建 `build.ps1`：

```powershell
$ErrorActionPreference = "Stop"
New-Item -ItemType Directory -Force -Path dist | Out-Null
$ldflags = "-s -w"

Write-Host "Building Windows..."
go build -ldflags=$ldflags -o dist/选股器.exe ./cmd/scanner

Write-Host "Building Linux..."
$env:CGO_ENABLED = "0"
$env:GOOS = "linux"
$env:GOARCH = "amd64"
go build -ldflags=$ldflags -o dist/scanner ./cmd/scanner
Remove-Item Env:GOOS, Env:GOARCH, Env:CGO_ENABLED

Write-Host "Done."
Get-ChildItem dist
```

## 5. 运行方式

### CLI 模式

```powershell
# 日线策略
.\选股器.exe -mode=day

# 周线策略 + 导出
.\选股器.exe -mode=week -export=json,md -out=./output

# 自定义参数（粘合度 1.2%，并发 80）
.\选股器.exe -mode=day -workers=80 -range=1.2 -bars=600
```

### HTTP 服务模式

```powershell
.\选股器.exe -serve -addr=:8080
```

访问 `http://localhost:8080/api/v1/health` 验证。

## 8. 常见问题

| 问题 | 处理 |
|------|------|
| 双击 exe 无反应 | CLI 程序需在命令行传参，如 `-mode=day` |
| 中文乱码 | PowerShell 执行 `chcp 65001`，或使用 Windows Terminal |
| 体积偏大 | Gin 依赖正常，约 10~15 MB；可用 UPX 进一步压缩 |
| 扫描超时 / 失败 | 检查网络能否访问 eastmoney.com |
| 端口占用 | 换 `-addr=:9090` 或其他端口 |

---

## 9. 可选：UPX 压缩

```powershell
upx --best dist/选股器.exe
```

可将体积再压缩约 50%，部分杀毒软件可能误报，需自行权衡。
